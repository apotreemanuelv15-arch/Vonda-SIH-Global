# VONDA SIH - Global Health Information System

**VONDA SIH** est un Système d'Information Hospitalier (SIH) de nouvelle génération, conçu pour optimiser la gestion médicale et administrative des structures de santé dans les zones à ressources limitées ainsi qu'à l'international.

## 🌟 Fonctionnalités Clés
- **Suivi Maternel Avancé (CPN) :** Calcul automatique de la DPA, suivi des constantes et alertes de risques.
- **Gestion des Pathologies Chroniques (VIH/SIDA, Diabète) :** Suivi de l'observance thérapeutique et des protocoles de soins.
- **Système de Notification Proactif :** Rappels de rendez-vous automatisés via SMS/Email pour réduire les perdus de vue.
- **Reporting Épidémiologique :** Génération automatique de rapports conformes aux normes de l'OMS et du SNIS.
- **Gestion Financière & Pharmacie :** Traçabilité totale des recettes et des stocks de médicaments.

## 🛠️ Stack Technique
- **Backend :** Python 3.x / Django (Architecture Robuste)
- **Base de données :** PostgreSQL / SQLite
- **Interface :** HTML5 / CSS3 / JavaScript (Design adaptatif)
- **Sécurité :** Cryptage des données patients et système de licence par "Gatekeeper".

## 🚀 Vision du Projet
Développé par **Emanuel VONDA**, ce logiciel vise à combler le fossé numérique dans le secteur de la santé, en offrant un outil qui n'est pas seulement une base de données, mais un véritable assistant médical intelligent.
